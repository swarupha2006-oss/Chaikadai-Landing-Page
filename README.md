# Chaikadai-Landing-Page
A responsive Tamil Street Food delivery landing page built with HTML and CSS
This project was developed as a learning exercise. I utilized Gemini (AI) to help me:
Understand responsive design principles (Flexbox and Grid).
Brainstorm a Tamil street-food themed color palette and UI layout.
Debug CSS path issues and GitHub deployment steps.
All code was reviewed, integrated, and tested by me to ensure it met the project requirements.
